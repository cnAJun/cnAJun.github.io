查看node和npm的版本，Linux版本号：

切换到root用户,安装 Hexo
sudo su
npm install hexo-cli -g（此过程大概需要五分钟）
安装成功
这里写图片描述

创建一个文件夹用于存放文章与hexo配置文件
mkdir hexo
这里写图片描述

然后完成初始化工作(此过程大概需要五分钟)
cd hexo
hexo init
这里写图片描述
初始化之后的目录结构：
这里写图片描述

安装npm依赖包，运行命令(在/hexo目录下)
cd hexo
npm install
这里写图片描述

接下来将静态页推送到托管平台上
安装hexo git插件，在hexo目录下运行命令
cd hexo
npm install hexo-deployer-git –save (此过程大概需要三分钟)

这里写图片描述

打开_config.yml文件，修改一下最下面的deploy：
这里写图片描述

改为

deploy:
type: git
repo: https://用户名:密码@仓库地址
branch: master
message: blog update

我的仓库地址：git.coding.net/dibage/dibage.git

然后推送到coding.net
hexo d

接下来就可以在项目的代码托管页看到推送过来的文件了
